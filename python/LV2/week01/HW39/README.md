建構唯一二元樹 

給定前序或後序以及中序，產生唯一個Binary Tree，依序印出Tree的內容，印出順序，由上而下，由左而右印出。 

前序代碼：P 
中序代碼：I 
後序代碼：O 

-------------- 
輸入說明 

第一筆輸入前序、中序或後序代碼。 
第二筆輸入上一筆輸入種類尋訪的結果，大寫英文字母。 
第三筆輸入前序、中序或後序代碼。 
第四筆輸入上一筆輸入種類尋訪的結果，大寫英文字母。 

-------------
輸出說明 

輸出為一二元樹的內容，由上而下，由左而右。 

------------- 
Sample input 

P 
ABCDEFGHI 
I 
BCAEDGHFI 

-------------- 
Sample output 

ABDCEFGIH
