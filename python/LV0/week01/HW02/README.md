# 一元二次方程式 I (Python_hw02)


>一元二次方程式，aX^2 + bx + c = 0，輸入a, b, c, 求 方程式的兩個實根。 

----------- 
##### 輸入說明 

第一個數 a  
第二個數 b  
第三個數 c 

--------------- 
##### 輸出說明 

第一個實根， x1 = ((-b)+sqrt(b*b-4*a*c))/(2*a) 
第二個實根， x2 = ((-b)-sqrt(b*b-4*a*c))/(2*a) 

> x1, x2 輸出到小數點第一位 

--------------- 
#### Input 
          1 
          -2 
          1 

#### Output

          1.0 
          1.0
